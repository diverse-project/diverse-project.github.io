As a Software Engineering team, DiverSE team produces a lot of code of various maturity. This goes from platform code to prototypes.

In order to keep track of such codes, this site lists the code repositories (or projects) and the related continuous integration servers.

Even if it is hard to maintain, this list is supposed to be exhaustive. So, if you are part of the Inria DiverSE team and your are working on a repository not listed here, please go to the github repository hosting this site and add a reference to your repository in the correct category.

As a Software Engineering team, DiverSE team produces a lot of code of various maturity. This goes from platform code to prototypes.

In order to keep track of such codes, this site lists the repositories (or projects) and the continuous integration servers that may be related to these repositories.

Even if it is hard to maintain, this list is supposed to be exhaustive. So, if you are part of the team and your are working on a repository not listed here, please go to the github repo of this site and add it in the correct category.
